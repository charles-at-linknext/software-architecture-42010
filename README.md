## 1. Introduction

本文討論一個 Healthcare 的物聯網系統架構，此架構可應用在 Healcare 有關的系統開發應用。本文目標是描述一份 best practice 與 methods，並提出一個 abstraction layers 設計。

本架構內容 components 都可以易於重用，同時也著試制定 component 的介面標準。
